# fablity

Fablity is primarily distributed as a Claude Code skills-directory plugin.
When installed with `npx skills add`, Claude Code loads the bundled protocol
skills, verifier agents, and hooks from this directory.

If this top-level skill is invoked directly, read `kernel/kernel.md` and follow
the fablity discipline kernel. Use the bundled protocol skills in `skills/`
for task-specific workflows, especially `spec-first` before multi-step
implementation and `fresh-verify` before completion claims.
